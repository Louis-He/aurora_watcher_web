# Aurora Watcher

This is the homepage for the iOS app Aurora Watcher - your ultimate companion for tracking and predicting aurora borealis activity.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
